# Privacy Policy (Summary)

This page provides a summary of how RondSync handles user data.

---

## Data Collected

- Wallet addresses  
- Transaction-related data  
- Platform usage data  
- Device and browser information  

---

## Key Principles

- Data is used to operate and improve the platform  
- Blockchain data is public and immutable  
- Personal data is not sold  

---

## Data Usage

Information is used for:

- Service operation  
- Security and fraud prevention  
- System improvement  
- Legal compliance  

---

## Important Notes

- Blockchain transactions are permanent  
- Wallet addresses may be considered personal data  
- Users should understand transparency risks
